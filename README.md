# koinara-app

User-facing client application and official website for the Koinara network.

## Workspaces

- `shared`: protocol-compatible types and utilities
- `app`: Electron + React desktop client
- `site`: static Vite + React website
- `config`: chain profiles consumed by app and site

## Scripts

Use `npm.cmd` on restricted PowerShell setups:

```powershell
npm.cmd install
npm.cmd run build
npm.cmd run test
```
